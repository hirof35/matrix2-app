# 行列による一次変換の可視化 (Matrix Transformation Visualizer)

C++とDXライブラリ（DxLib）を使用して、2次元の行列（一次変換）による図形の変形をリアルタイムに視覚化するアプリケーションです。

##  概要
数学における「行列による座標変換」を直感的に理解するためのツールです。
画面上の入力キーを操作することで、行列 $M = \begin{pmatrix} a & b \\ c & d \end{pmatrix}$ の各成分をリアルタイムに変更し、基準となる図形（緑の三角形）がどのように変形（拡大・縮小、回転、せん断など）して新しい図形（青の三角形）に写るかのアニメーションを確認できます。

## ⚙️ 動作環境
- **OS:** Windows 10 / 11
- **開発環境:** Visual Studio 2022 以降推奨
- **グラフィックライブラリ:** DXライブラリ (DxLib) Ver 3.24 以降

## 🎮 操作方法
キーボードの各キーを押すことで、行列の成分（初期値は単位行列）を `0.02` ずつ増減させることができます。

| 行列の成分 | 増加キー | 減少キー |
| :---: | :---: | :---: |
| **a** (X軸のXスケール) | `Q` | `A` |
| **b** (X軸へのYの傾き) | `W` | `S` |
| **c** (Y軸へのXの傾き) | `E` | `D` |
| **d** (Y軸のYスケール) | `R` | `F` |

* **ウインドウの閉じるボタン** または `Esc` キーでアプリケーションを終了します。

## 📐 変換のロジック
元の座標を $(x, y)$、変換後の座標を $(x', y')$ としたとき、プログラム内部では以下の行列計算を毎フレーム行っています。

$$
\begin{pmatrix} x' \\ y' \end{pmatrix} = \begin{pmatrix} a & b \\ c & d \end{pmatrix} \begin{pmatrix} x \\ y \end{pmatrix}
$$

つまり、プログラム内の計算式は以下の通りです：
* $x' = a \cdot x + b \cdot y$
* $y' = c \cdot x + d \cdot y$

## 📂 構築・実行手順
1. Visual Studio で「空のC++プロジェクト」を作成します。
2. DXライブラリの公式サイトの手順に従い、プロジェクトの「インクルードディレクトリ」と「ライブラリディレクトリ」を通します。
3. `main.cpp` を作成し、ソースコードを貼り付けます。
4. プロジェクトのプロパティから、構成システムを **「Windows (/SUBSYSTEM:WINDOWS)」** に設定します。
5. ビルドして実行（`F5`）します。
